# Habits App\n\nA full-stack application with a FastAPI backend and a React/Vite frontend.\n\n## Project Structure\n\n- `backend/`: FastAPI application.\n- `frontend/`: React/Vite application.\n- `nginx.conf`: Reverse proxy configuration.\n- `docker-compose.yml`: Docker orchestration.\n- `Makefile`: Utility commands for development and testing.\n\n## Getting Started\n\nUse the `Makefile` to set up and run the application:\n\n```bash\nmake setup\nmake dev-back\nmake dev-front\n```\n\nOr use Docker:\n\n```bash\ndocker-compose up --build\n```
